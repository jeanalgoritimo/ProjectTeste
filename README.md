# Instrução de teste da aplicação ApiTeste é ProjectTeste
Necessário entrar com botão direito da solução, seguir para Properties é ao abrir a janela escolher a opção em Common Properties
na Startup Projet o Radio button Multiple startup projects ApiTeste escolher ação Start,ProjectTeste escolher ação Start e assim fechar a janela.
Logo em sequida selecionar o botão verde de start do visual studio esperar os serviçõs inciarem.
O Site será iniciado no local host no navegador no endereço https://localhost:44401/ é a api abrirá no local host 
no navegador no endereço https://localhost:44365/
